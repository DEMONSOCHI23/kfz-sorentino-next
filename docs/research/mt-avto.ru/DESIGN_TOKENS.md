# mt-avto.ru Approximate Design Tokens

## Colors

- Page background: `#f5f2ec`
- Surface: `#ffffff`
- Dark surface: `#171412`
- Primary red: `#bc2d23`
- Secondary beige: `#efe4d5`
- Outline: `#d6c8b6`
- Gold accent: `#c5a26a`

## Typography

- Primary style: geometric sans similar to Montserrat / Arial fallback
- Headline tracking: tight, usually `-0.03em` to `-0.05em`
- Eyebrow labels: uppercase with wide tracking
- Body copy: 14px to 18px, relaxed line height

## Radius / Space

- Major cards: `28px` to `36px`
- Pills / buttons: fully rounded
- Section shell: `1280px` max width
- Grid gaps: `16px` to `20px`
