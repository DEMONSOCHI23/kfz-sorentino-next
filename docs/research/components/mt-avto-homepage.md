# Component Spec: MT-AVTO Homepage Clone

## Goal

Reconstruct the main-page layout language of `mt-avto.ru` without copying live branding verbatim.

## Section Specs

### Sticky Header
- Dark translucent bar
- Compact icon box + uppercase brand line
- Desktop nav hidden on smaller screens
- Right-aligned rounded booking CTA

### Hero
- Desktop split layout: large dark left panel, stacked cards on right
- Large display heading, short subcopy, 3 metric cards
- Right column includes toggle pills and a dark CTA card
- Secondary red and white info cards below

### Services Grid
- 4-column desktop, 2-column tablet, 1-column mobile
- White rounded cards with index number and arrow/icon affordance

### Dark Advantage Section
- Strong contrast band with stats on dark background
- Adjacent vertical stack of white feature cards

### Testimonials
- Three white cards, soft shadows, red star row accent

### Contact / Route Section
- Red info block on left
- Split route block on right with dark intro panel and light map placeholder
