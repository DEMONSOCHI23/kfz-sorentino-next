# mt-avto.ru Behavior Notes

- Header is fixed/sticky and sits on top of the first screen.
- Large rounded cards dominate the layout; hover states are subtle translate/shadow changes rather than heavy animation.
- Main page behavior is mostly static with strong CTA emphasis, not app-like interaction.
- Several sections rely on contrast shifts: light page -> dark highlight section -> light trust section -> bold contact CTA.
- Mobile layout collapses the multi-column desktop composition into stacked cards with preserved border radii.
