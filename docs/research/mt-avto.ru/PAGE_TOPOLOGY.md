# mt-avto.ru Page Topology

1. Sticky header with compact branding, short nav, and booking CTA.
2. Hero area with one dominant dark panel and a secondary right-side card stack.
3. Dense service grid made of repeated white rounded cards.
4. Dark contrast section with metrics and value props.
5. Testimonial / trust band with three review cards.
6. Closing contact block with route/map presentation and final CTA.

Interaction model:
- Header: sticky, static links.
- Hero: lightweight pill/toggle interaction inside the right card stack.
- Service cards: hover affordance only.
- Remaining sections: static, scroll-driven reveal only via layout sequencing.
